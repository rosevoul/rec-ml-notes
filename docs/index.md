# Machine Learning and GenAI for Recommendation Systems - Notes

These notes document ongoing exploration of selected modeling approaches
commonly used in modern personalization systems.

The goal is to build working intuition around how different methods integrate
into real-world ranking and recommendation pipelines.

## Topics
- Graph-based recommendation
- LLM-based reasoning in ranking pipelines
- Multimodal recommendation
- Generative sequence-based recommendation
